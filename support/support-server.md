Join the [Afterparty Bot Labs support server]({{ site.links.support }}) for NovaPulse assistance.

## Support can help with

- Installation and missing slash commands.
- Discord permissions and role hierarchy.
- Configuration channels and roles.
- Verification workflow questions.
- Controlled NovaTrap or anti-nuke testing questions.
- Safe bug reports and error reference IDs.
- Privacy or deletion requests.
- False-positive escalation.

## Support cannot safely use

Do not provide credentials, complete private configuration, production databases, raw runtime logs, identity documents, or an environment file. Staff should ask for the smallest safe set of IDs and reproduction details.

## Response expectations

Support availability is not guaranteed to be uninterrupted. Provide a concise report and avoid duplicate requests. For immediate server safety, disable the affected feature and use Discord's native moderation controls while waiting.
