## About

NovaPulse helps Discord communities organize moderation, onboarding, verification, logging, automation, and server-security workflows.

## Core features

- Moderation and staff administration tools
- Server-owned activity and moderation logging
- Configurable automation and role workflows
- Verification and onboarding guidance
- NovaTrap and server-security tooling
- Server configuration, permissions, and troubleshooting guides
